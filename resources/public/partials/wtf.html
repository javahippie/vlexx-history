Was soll das?

Was passiert auf dieser Seite?

Dieses Programm verfolgt Verspätungszeiten der Vlexx-Züge am Bahnhof Mainz. Ziel ist es, neben einer aktuellen Übersicht und einer Tagesübersicht auch langfristig Statistiken aufzubauen.

Warum nur in Mainz?

Ich frage die Daten von der offiziellen Vlexx-Webseite ab. Hier die Daten für jeden Bahnhof anzufragen würde eine höhere Serverlast bedeuten, weshalb ich (zumindest zur Zeit) davon absehe. Nennen wir es Etiquette.

Und warum tust du das?

Aus mehreren Gründen. Da das Vlexx Verkehrsunternehmen einen holprigen Start auf der Strecke hingelegt hat (und alle sich darüber aufgeregt haben), hat es mich interessiert, wie es denn mit den Verspätungszeiten wirklich aussieht. Außerdem hat es mir die Möglichkeit gegeben, mich beim Programmieren an ein paar neuen Techniken abzuarbeiten.

Welche Techniken werden eingesetzt?

Das Backend läuft auf Clojure und einer MongoDB, das Frontend wurde mit AngularJS umgesetzt. Wenn du dich für den Code interessierst, kannst du dich gerne auf meinem GitHub umsehen.
